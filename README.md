# keyrotator
This is a CLI utility to help with key rotation.

## Install

Install with

```sh
$ python setup.py install
```

## Uninstall

Uninstall with

```sh
$ pip uninstall keyrotator
```

## Usage

From the command line, run:

```sh
$ keyrotator
```
